
# 🔋 Battery Fault Detection System

End-to-end signal processing + ML pipeline for battery fault detection.

## Features
- Signal processing (FFT, STFT, Wavelet)
- Feature engineering
- Baseline health detection
- Random Forest fault prediction

## Run
```bash
pip install -r requirements.txt
python src/model_training.py
```

## Notes
- Uses sample battery signal data
- Outputs results to outputs/
